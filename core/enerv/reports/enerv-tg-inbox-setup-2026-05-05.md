### 5. Технические решения

- **Shell escaping**: Code node делает `'` → `'\''` перед передачей JSON как shell argument
- **Gemini fallback**: Code node пробует прямой JSON.parse, fallback — regex strip на ```code block```
- **chatId propagation**: Code node захватывает `$('Telegram Trigger').first().json.message.chat.id` и прокидывает через все nodes

## Обнаруженные проблемы (для будущих сессий)

### n8n Session Cookie Issue
- MCP browser (chrome-devtools-mcp) не держит n8n session cookie между навигациями
- Cloudflare блокирует direct external API calls (Python/curl) с 403
- **Workaround**: использовать `evaluate_script` для fetch calls из browser context (same-origin)
- **Рекомендация**: сохранить `ENERV-CLAUDE` API key в `~/.hermes/.env` как `N8N_API_KEY` для future automation

### Google OAuth vs API Key
- Пользователь предоставил Google Cloud OAuth credentials (client_id/secret) — они для OAuth2 flow
- Gemini HTTP Request node использует `httpQueryAuth` (simple API key)
- Решение: пользователь создал отдельный Google API Key (`AIza...`) в GCP Console
