## 🚀 Overview
**SOVRN** is a sovereignty-first personal AI organization stack. It enables a Solo Vibe Coder to orchestrate a resilient, local-first intelligence network.

- **Boss:** Hermes Agent (WSL2 / Linux)
- **Sub-Agents:** Aider, Cline, Claude Code
- **Model Gateway:** LiteLLM (Local llama.cpp + Cloud Pool)
- **Memory:** PARA (Phase 0) → Graphiti (Phase 1)
- **Observability:** Langfuse + OpenLLMetry
